PROJECT CHIMERA
Design System · v0.1

A dark-first, faceted, cel-shaded UI language built for clarity over a busy 3D battlefield — with a warm-paper light variant for the editor crowd. Flat color blocks, crisp chamfered edges, a thin bright edge-light on raised panels, geometric iconography, and a single reserved accent. Every component maps to a standard Godot Control node. Switch the accent or theme above to compare options live.

07

Faceted Panel Anatomy

panel — primary

Chamfered top-left / bottom-right corners. Faceted hairline border with a brighter cel-shade top edge. Used for all docked surfaces.

panel--2 — raised card

Lighter surface for cards and nested groups. Same facet geometry, one step up in elevation.

panel--accent — focus

Accent-lit border for the active / selected panel — e.g. the hero Unit Card under edit.

Chamfers are kept small (5–14px) and applied to two opposite corners — enough to read as faceted/low-poly without the angularity ever competing with text. Borders are crisp 1px gradients (cel edge-light → shadow), never glows on dense surfaces. No per-frame blur on HUD panels; blur is reserved for paused modal scrims.
