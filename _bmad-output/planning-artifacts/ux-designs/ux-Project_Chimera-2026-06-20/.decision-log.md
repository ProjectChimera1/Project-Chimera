## D2 — Design language (from `chimera.css`)
- **Low-poly echo, faceted dark UI.** Chamfered corners via `clip-path` polygons (`--cut` 8/5/14px), thin cel-shade top-edge highlight, flat color blocks — UI chrome echoes the low-poly 3D art so UI + world read as one piece.
- **HUD is non-diegetic** flat overlay (SC2/classic-RTS convention) — clarity over in-world illusion.
- **Accent = teal default**, with `amber` and `violet` accent variants (`[data-accent]`). Dark theme default; a full **light theme** (warm paper) exists (`[data-theme="light"]`).
- **Type:** Chakra Petch (display/headings/buttons, uppercase + letter-spacing), Space Grotesk (UI/body), JetBrains Mono (numbers/readouts, tabular-nums). 1.250 scale, 11→72px.
- **Faction/team colors are sacred + reserved** — 8 **Okabe-Ito-derived, colorblind-safe** team colors live on units; UI accents must never be confused for them.
- **Chimera Seal mark** — geometric alchemical "transmutation sigil" (two rings + fire/water triangles + nucleus) + a **transmute spinner** loading indicator. Monochrome, engraves on any surface.
